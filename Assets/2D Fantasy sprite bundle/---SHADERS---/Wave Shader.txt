Wind Speed ​​- Wave speed
Wind Direction - Wave direction
WindScale - Wave size
WindStreght - Wave strength

Grounded - If true, then the wave affects the upper part of the sprite, but does not affect the lower part (Can be used for grass).

InvertGround - If true, then the wave does not affect the upper part of the sprite, but it does affect the lower part (Can be used for plants that hang down).

WindInfluenceMask - If 0 - grounded does not affect, if >0 affects
